What this is

A practical guide to how we work as a team: how we onboard, communicate, run projects, make decisions, and keep the work moving.

This is where we document the habits, rituals, standards, and expectations that shape day-to-day life at TRIPTK. It covers how people get started, how projects are set up and reviewed, how feedback works, and how we make sure the work stays sharp.

Use it when you’re new to the team, joining a project, or trying to understand how something gets done. It’s part process, part shared standard, and it should keep getting better as we do.
